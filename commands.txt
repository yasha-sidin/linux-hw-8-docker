Настраиваем конфигурацию веб-сервера:

    mkdir wordpress && cd wordpress

    mkdir nginx-conf

    vim nginx-conf/nginx.conf

    Добавляем в файл конфигурацию(Находится в файле nginx.conf в данном репозитории) и сохраняем

Настраиваем переменные среды:

    vim .env

    Добавляем в файл конфигурацию со своими значениями(Пример находится в файле .env в данном репозитории) и сохраняем

    git init

    vim .gitignore(Добавляем в файл .env)

    vim .dockerignore(Добавляем в файл .env)

    touch docker-compose.yml

    git add .

    git commit -m "Some text"

    vim docker-compose.yml
